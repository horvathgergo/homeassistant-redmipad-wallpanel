# Redmi Pad SE Wall Panel for Home Assistant
This project shows how I built a wall-mounted Home Assistant dashboard using a Redmi Pad SE 11 tablet and an IKEA RÖDALM picture frame.

![Image](https://github.com/user-attachments/assets/66a884fb-16db-441f-be6d-f404e449e3cc)

I used the **11-inch version of the Redmi Pad SE**, which fits nicely inside the **21 × 30 cm, 3 cm deep RÖDALM frame**.  
However, the cardboard front of the frame had to be **trimmed slightly to match the tablet size**.

To mount the frame securely on the wall, I also used **two small wall hooks purchased from AliExpress**.
https://a.aliexpress.com/_EIA3dbS
![Image](https://github.com/user-attachments/assets/3b70605a-edc0-4438-9502-0a4d608bb8b7)

The result is a clean and simple wall panel for Home Assistant, using inexpensive and easy-to-find components.

## Cost
- Redmi Pad SE 11 tablet – **100 EUR**
- IKEA RÖDALM picture frame (21×30 cm) – **5 EUR**
- Wall hooks (AliExpress) – **3 EUR**
- Charger cable – **4 EUR**
- Sonoff ZBMINIR2 smart relay – **10 EUR**

**Total: ~122 EUR**
